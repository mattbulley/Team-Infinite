These instructions describe how to setup and run a local web server.

First time setup:
1. Install Node.js: https://nodejs.org/en/
2. Install http-server by opening the command prompt (Windows key + R, then type 'cmd' and press enter).
   In the command prompt type: 'npm install -g http-server'

Run the web server:
3. Open the command prompt (see step 2) and type: 'http-server'
   Open your web browser and go to: 'localhost:8080'
4. To close the web server, in the command prompt press: 'ctrl + C'
